## Project Description

This project is a simple 2D vector drawing application developed using Python, PyOpenGL, and Pygame. Users can draw lines and polygons, select shapes, and apply affine transformations such as translation, rotation, and scaling.

## Features

- Draw lines
- Draw polygons
- Select shapes
- Translate selected shapes
- Rotate selected shapes
- Scale selected shapes
- Delete selected shapes
- Clear the canvas

## Controls

| Key | Function |
|-----|----------|
| L | Line mode |
| P | Polygon mode |
| S | Select mode |
| Enter | Finish polygon |
| Arrow Keys | Move selected shape |
| R | Rotate selected shape |
| + | Scale up |
| - | Scale down |
| Delete / Backspace | Delete selected shape |
| C | Clear canvas |

## Requirements

- Python 3
- Pygame
- PyOpenGL

## Run
```bash
pip install -r requirements.txt  
python main.py
```
